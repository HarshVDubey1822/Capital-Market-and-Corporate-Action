# 📊 Capital Markets & Corporate Actions Data Analysis

## 🔍 Project Overview

This project focuses on analyzing capital market data and corporate actions to generate actionable financial insights. It simulates real-world workflows used by financial analysts in investment firms and market data platforms such as Bloomberg and FactSet.

The analysis covers equities, derivatives, and bond instruments, along with corporate events including dividends, stock splits, and mergers.

---

## 🎯 Objectives

* Analyze 10,000+ capital market records
* Identify price trends and volatility patterns
* Perform data cleaning and anomaly detection
* Conduct pricing validation using SQL
* Analyze the impact of corporate actions on returns and trading volume
* Develop an interactive Power BI dashboard for financial insights

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL (SQLite)**
* **Power BI**
* **Excel**
* **AI-assisted Analytics (ChatGPT)**

---

## 🔄 Project Workflow

### 1. Data Preparation

* Imported and structured raw financial datasets
* Handled missing values using forward-fill techniques
* Standardized date formats and column names

### 2. Exploratory Data Analysis (EDA)

* Analyzed price movements and trading volume
* Identified outliers and inconsistencies
* Generated summary statistics

### 3. Financial Metrics Calculation

* Daily Returns
* Moving Averages (20-day, 50-day)
* Volatility (20-day rolling standard deviation)
* Risk Index

### 4. SQL Data Validation

* Detected missing and invalid price values
* Identified duplicate records
* Performed anomaly detection and pricing validation

### 5. Corporate Actions Analysis

* Tracked 150+ corporate events
* Categorized actions (Dividends, Splits, Mergers)
* Measured impact on returns and trading volume

### 6. Dashboard Development (Power BI)

* Built interactive dashboard with financial KPIs
* Visualized trends, volatility, and corporate action impact
* Added filters for dynamic analysis

---

## 📊 Key Insights

* Market volatility increases during corporate action events
* Trading volume spikes align with dividend announcements
* Moving averages indicate trend reversals and price momentum
* Corporate actions significantly influence short-term returns
